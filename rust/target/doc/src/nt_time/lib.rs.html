5//! The `nt-time` crate is a [Windows file time] library.
6//!
7//! # Examples
8//!
9//! ## Basic usage
10//!
11//! The [`FileTime`] is a type that represents the file time. This type can be
12//! converted from and to a type which represents time such as
13//! [`time::OffsetDateTime`]. Addition and subtraction are also supported.
14//!
15//! ```
16//! use core::time::Duration;
17//!
18//! use nt_time::{
19//!     time::{macros::datetime, OffsetDateTime},
20//!     FileTime,
21//! };
22//!
23//! let ft = FileTime::NT_TIME_EPOCH;
24//! assert_eq!(
25//!     OffsetDateTime::try_from(ft).unwrap(),
26//!     datetime!(1601-01-01 00:00 UTC)
27//! );
28//!
29//! let ft = ft + Duration::from_secs(11_644_473_600);
30//! assert_eq!(
31//!     OffsetDateTime::try_from(ft).unwrap(),
32//!     OffsetDateTime::UNIX_EPOCH
33//! );
34//! assert_eq!(ft.to_raw(), 116_444_736_000_000_000);
35//!
36//! assert_eq!(FileTime::new(u64::MAX), FileTime::MAX);
37//! ```
38//!
39//! ## Conversion from and to other system times
40//!
41//! [`FileTime`] can be converted from and to other system times such as [Unix
42//! time] or [MS-DOS date and time].
43//!
44//! ```
45//! use core::time::Duration;
46//!
47//! use nt_time::{
48//!     time::{OffsetDateTime, UtcOffset},
49//!     FileTime,
50//! };
51//!
52//! // `1970-01-01 00:00:00 UTC`.
53//! let ut = i64::default();
54//! assert_eq!(
55//!     OffsetDateTime::from_unix_timestamp(ut).unwrap(),
56//!     OffsetDateTime::UNIX_EPOCH
57//! );
58//!
59//! let ft = FileTime::from_unix_time(ut).unwrap();
60//! assert_eq!(ft, FileTime::UNIX_EPOCH);
61//!
62//! // `1980-01-01 00:00:00 UTC`.
63//! let ft = ft + Duration::from_secs(315_532_800);
64//! let dos_dt = ft.to_dos_date_time(Some(UtcOffset::UTC)).unwrap();
65//! assert_eq!(dos_dt, (0x0021, u16::MIN, u8::MIN, Some(UtcOffset::UTC)));
66//! ```
67//!
68//! ## Formatting and printing the file time
69//!
70//! The formatting traits for [`FileTime`] are implemented to show the
71//! underlying [`u64`] value. If you need a human-readable date and time,
72//! convert [`FileTime`] to a type which represents time such as
73//! [`time::OffsetDateTime`].
74//!
75//! ```
76//! use nt_time::{time::OffsetDateTime, FileTime};
77//!
78//! let ft = FileTime::NT_TIME_EPOCH;
79//! assert_eq!(format!("{ft}"), "0");
80//! assert_eq!(
81//!     format!("{}", OffsetDateTime::try_from(ft).unwrap()),
82//!     "1601-01-01 0:00:00.0 +00:00:00"
83//! );
84//! ```
85//!
86//! [Windows file time]: https://docs.microsoft.com/en-us/windows/win32/sysinfo/file-times
87//! [Unix time]: https://en.wikipedia.org/wiki/Unix_time
88//! [MS-DOS date and time]: https://learn.microsoft.com/en-us/windows/win32/sysinfo/ms-dos-date-and-time
